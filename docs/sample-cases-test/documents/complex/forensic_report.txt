ROOT CAUSE ANALYSIS
Primary vector: Unpatched Palo Alto VPN (PAN-OS 10.1.6, CVE-2024-3400)
Contributing factors: No MFA on jmartinez_svc service account; MSP patch deferral;
  backup server on same VLAN as production; delayed SIEM alert tuning.

REMEDIATION RECOMMENDATIONS
1. Complete PAN-OS patch to 11.1.4-h1; enforce MFA on all VPN accounts within 72 hours
2. Segment backup infrastructure; implement PAM for database administrators
3. Deploy Canary tokens in CRM export directories
4. Terminate or renegotiate CloudSecure MSP contract; internal patch ownership
5. Customer notification: 847 individuals + 23 SSN supplemental state AG filing
6. Retain logs 365 days; engage SEC counsel if materiality threshold met ($1.2M impact est.)
